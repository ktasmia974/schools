## 📌 Overview

Complete **Data Analysis Assignment** on **20 leading schools from around the world** — including Eton College, Phillips Academy Andover, Le Rosey, Aitchison College, and Beaconhouse School System. Analyzes fees, student/teacher counts, campus size, and global rankings across 10 countries.

## 📊 Dataset Features

| Column | Description |
|--------|-------------|
| `School_Name` | Name of the school |
| `Founded` | Year founded |
| `City` / `Country` | Location |
| `Students` | Total enrolled students |
| `Teachers` | Number of teaching staff |
| `Type` | Public / Private |
| `Annual_Fee_USD` | Annual tuition fee in USD |
| `Campus_Acres` | Campus size in acres |
| `Global_Ranking` | Global ranking (1 = best) |
| `Extracurricular_Activities` | Number of clubs/activities offered |
| `Rating_Out_Of_5` | Overall rating |

## 🚀 How to Run

### Google Colab
1. Go to [colab.research.google.com](https://colab.research.google.com)
2. **File → Upload notebook** → select `School_Analysis.ipynb`
3. **Runtime → Run All**

### Local
```bash
pip install pandas matplotlib seaborn plotly
jupyter notebook School_Analysis.ipynb
```
