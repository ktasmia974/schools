Finds the minimum number of trials needed to find the critical floor with `n` eggs and `k` floors, using the recursive DP formulation.

## How to Compile & Run

```bash
g++ EggDroppingPuzzle.cpp -o eggdrop && ./eggdrop
```

### Expected Output

```
Minimum trials with 2 eggs and 10 floors: 4
```
